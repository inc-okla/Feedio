Flow Ultra

Flow Ultra adalah modul integrasi untuk menghasilkan konten visual berkualitas tinggi (Video & Image) menggunakan model generatif terbaru seperti Veo, Imagen 4, dan Nano Banana Pro.

Integrasi API & Token

Koneksi ke Feedio Bot wajib menggunakan Token API khusus yang didapatkan melalui ekstensi browser. Ikuti langkah berikut:

PENTING: Token ini hanya berlaku untuk pengguna Paket Flow Ultra.
  1. Unduh Ekstensi Feedio APIs di sini: Feedio.pro/APIs.
  2. Buka halaman Google Flow setelah ekstensi terpasang untuk mendapatkan token (Contoh: ya.29xxxxddsg...).
  3. Salin token tersebut. Di aplikasi Feedio, klik logo Kunci (API Access).
  4. Tempel token di kolom API Key agar bot terhubung secara otomatis.

Remove Background & Upscale

Modul ini ditenagai oleh infrastruktur AI canggih yang dikelola oleh Feedio. Dirancang khusus untuk efisiensi tinggi, fitur ini memungkinkan pemrosesan gambar secara massal (Batch Processing) dengan hanya mengandalkan koneksi jaringan yang stabil.

Semua pemrosesan dilakukan secara cloud-based (online), sehingga tidak membebani kinerja perangkat keras (CPU/GPU) komputer Anda.

Fitur Utama

  • Remove Background: Menghapus latar belakang dengan presisi tinggi.
  • Image Upscale: Meningkatkan resolusi gambar hingga 2X lipat menggunakan teknologi AI, menjadikan gambar lebih jernih dan detail.

Mekanisme Validasi File & Efisiensi Kuota

PENTING: Logika Pemrosesan Otomatis

Sistem Feedio menerapkan protokol efisiensi cerdas untuk melindungi kuota API dan mencegah duplikasi pekerjaan. Harap perhatikan mekanisme berikut:

  1. Penamaan Otomatis (Auto-Renaming): Setiap file sumber yang berhasil diproses akan secara otomatis diubah namanya dengan menyisipkan label Feedio_Success pada nama file.
  2. Pencegahan Proses Ganda: Sistem diprogram untuk secara otomatis melewati (skip) file apa pun yang mengandung nama Feedio_Success. Ini memastikan file yang sama tidak diproses ulang secara tidak sengaja.
  3. Indikator Proses Berhenti: Jika setelah memasukkan API Key proses langsung berhenti tanpa memproses satu pun file, mohon periksa folder sumber Anda. Hal ini menandakan bahwa seluruh file di dalamnya telah teridentifikasi sebagai file yang sudah selesai diproses (memiliki label success).
  4. Instruksi Pemrosesan Ulang: Jika Anda bermaksud memproses ulang file yang sudah pernah sukses, Anda wajib melakukan Rename manual pada file tersebut untuk menghilangkan kata Feedio_Success sebelum memulai proses kembali.
TIPS OPTIMALISASI HASIL:
  • Background: Disarankan menggunakan gambar dengan objek yang jelas dan latar belakang datar/sederhana (tidak terlalu ramai) untuk hasil pemotongan yang presisi.
  • Ukuran File: Maksimal ukuran file yang dapat diproses adalah 23 MB per gambar.
  • Urutan Proses: Sangat disarankan untuk melakukan Upscale terlebih dahulu, baru kemudian melakukan Remove Background untuk detail yang lebih baik.

Topaz Upscaler

Modul Topaz Upscaler di Feedio memungkinkan Anda melakukan restorasi dan peningkatan kualitas visual tingkat bioskop secara massal (Batch Processing). Dengan teknologi AI canggih, Anda dapat meningkatkan video hingga resolusi 4K dan gambar hingga resolusi tinggi dengan detail yang super tajam.

Sistem ini berjalan sepenuhnya secara Online (Cloud-Based). Artinya, Anda dapat memproses video 4K bahkan pada perangkat dengan spesifikasi rendah ("device kentang") tanpa membebani hardware komputer Anda.

Fitur & Konfigurasi

1. Mode & Pengaturan (Settings)

Di menu Home, tersedia dua pilihan model utama:

  • Topaz Gigapixel: Peningkatan kualitas untuk foto.
  • Topaz Video: Peningkatan kualitas untuk video.

Pengaturan detail di menu Settings (Gerigi):

  • Resolusi (Resolution): Pilihan peningkatan HD, 4K, atau skala 2x dan 4x.
  • Model AI: Tersedia berbagai model Topaz yang dapat disesuaikan dengan jenis footage.
  • Video Settings:
    • Audio: Pilihan Copy (pertahankan suara) atau None (hapus suara).
    • FPS: Tingkatkan kelancaran video ke 24, 30, atau 60 FPS.

2. Otomatisasi API Key (Software Feedio APIs)

Untuk menggunakan fitur ini, Anda memerlukan API Key khusus Topaz. Feedio menyediakan alat bantu otomatis berupa software desktop "Feedio APIs" yang 100% Gratis (selama kebijakan Topaz mengizinkan).

Kapasitas API: 1 (Satu) API Key biasanya dapat memproses hingga 7 Video (Resolusi 4K) sebelum expired. API tidak memiliki batasan waktu (no expiry time) jika belum digunakan.
  1. Unduh dan instal software Feedio APIs dari tautan di atas.
  2. Buka software tersebut, lalu klik tombol Start.
  3. Software akan secara otomatis menghasilkan API Key Topaz sebanyak mungkin tanpa campur tangan manual.
  4. Salin API Key yang dihasilkan.

3. Batch API Processing (Paralel)

Salah satu fitur unggulan Feedio adalah kemampuan memproses banyak API sekaligus (Batch API). Ini memungkinkan proses upscale berjalan secara paralel dan berkelanjutan.

Masuk ke menu API Access (Logo Kunci) di Feedio, lalu masukkan API Key Anda dengan format satu key per baris (enter ke bawah).

Contoh Format Single API Key:
b8e7ea1f-5115-49b7-80a7-11b9ebe6a7ea
Contoh Format Batch API Key (Susunan Multi-line):
b8e7ea1f-5115-49b7-80a7-11b9ebe6a7ea
3a1cc23a-b2de-4497-a4cf-32787a305f8a
096378cb-394a-4362-a406-a78f16d4de50
1b068250-eba7-4579-9244-a1e5922ed004

Dengan memasukkan banyak API sekaligus, Feedio akan otomatis beralih ke API berikutnya jika API sebelumnya sudah mencapai batas limit (7 video), memastikan antrian render Anda tidak terputus.

Mekanisme Validasi File & Efisiensi Kuota

PENTING: Logika Pemrosesan Otomatis

Guna menghemat penggunaan API Key Topaz dan waktu pemrosesan, sistem menerapkan aturan validasi berikut:

  1. Penamaan Otomatis (Auto-Renaming): Setiap file yang telah selesai diproses akan di-rename oleh sistem dengan penambahan akhiran Feedio_Success.
  2. Deteksi File Selesai: Sistem akan memindai folder input dan secara otomatis melewati (skip) file yang memiliki nama Feedio_Success agar tidak membuang kuota API pada file yang sama.
  3. Indikator Berhenti Otomatis: Apabila Anda menekan tombol proses dan sistem langsung berhenti tanpa aktivitas, harap periksa folder sumber. Hal ini terjadi karena sistem mendeteksi semua file dalam folder tersebut sudah pernah diproses sebelumnya.
  4. Cara Memproses Ulang: Jika Anda ingin melakukan upscale ulang pada file yang sama, Anda harus me-rename file tersebut kembali ke nama aslinya (hapus bagian Feedio_Success) agar sistem membacanya sebagai file baru.

Kode Kesalahan (Error Codes)

400 - Bad Request Permintaan berisi data yang tidak valid, format salah, atau parameter query yang tidak dikenali server.
401 - Unauthorized Otorisasi gagal. Kunci API (API Key) tidak valid, kosong, atau telah kadaluwarsa.
403 - Forbidden Akses ditolak. Kunci API Anda tidak memiliki izin untuk permintaan ini atau kuota telah habis.
404 - Not Found Sumber daya atau endpoint yang diminta tidak ditemukan di server Topaz.
409 - Conflict Terjadi konflik pemrosesan permintaan dengan keadaan sumber daya server saat ini.
413 - Payload Too Large Ukuran file input terlalu besar dan melebihi batas kapasitas pemrosesan server.
429 - Too Many Requests Rate Limiting terlampaui. Terlalu banyak permintaan dikirim dalam waktu singkat. Silakan ganti API Key.
500 - Internal Server Error Terjadi kesalahan teknis yang tidak terduga pada sisi server Topaz. Silakan coba beberapa saat lagi.
