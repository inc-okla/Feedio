Feedio APIs – Privacy Policy

Last updated: February 2025

Feedio APIs (“the Extension”) is a browser extension that helps users generate API structures and keys using AI tools. This Privacy Policy explains what data the Extension collects, how it is used, and how it is protected.

1. Data We Collect

The Extension does not collect or store any personally identifiable information (PII). The only data stored locally in the browser is:

  • User preferences such as selected AI model
  • Previously generated API parameters

2. How We Use Data

Locally stored data is used only to:

  • Improve user experience
  • Remember extension settings
  • Support AI-based API generation

3. Host Permissions

The extension uses host permissions for:

  • https://firefly.adobe.com/
  • https://labs.google/

These permissions are only used to enable communication between the extension UI and these pages. The extension does not read or collect any content from those websites.

4. Third-Party Services

The Extension may connect to external AI services exclusively to generate API content. These requests are initiated only by the user. We do not receive or store data from these services.

5. No Remote Code

The Extension does not load or execute remote JavaScript. All logic is packaged inside the extension.

6. Data Sharing

We do not share, sell, or transfer any data to third parties. No data leaves the user’s device unless the user triggers an API request.

7. Security

All data is stored locally and protected by Chrome’s extension sandbox.

8. Children’s Privacy

The Extension is not intended for children under 13.

9. Changes to This Policy

We may update this Privacy Policy. The latest version will always be available here.
