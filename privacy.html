Privacy Policy

Feedio APIs — Browser Extension

Last updated: February 2025

Feedio APIs (“the Extension”) is a browser tool that helps users generate API structures using AI services. This Privacy Policy explains how the Extension uses and protects your information.

1. Information We Collect

The Extension does not collect or store any personally identifiable information (PII).

The only data stored is:

  • Preferred AI model
  • Previously generated API parameters

2. How Your Information Is Used

All data is stored locally on your device and is used only to enhance user experience within the extension.

3. Host Permissions

The Extension interacts only with the following domains:

  • https://firefly.adobe.com
  • https://labs.google

These permissions allow the Extension to attach a small script so it can generate API information. We do not collect or read any content from these websites.

4. External APIs

The Extension may send data to AI services only when the user initiates an API generation request. No information is transmitted automatically.

5. No Remote Code

The Extension does not load or run remote JavaScript. All logic is packaged inside the extension files.

6. Data Sharing

Feedio APIs does not sell, share, or transmit user data to third parties.

7. Security

All stored information remains on the user's device and is protected by browser sandboxing.

8. Children's Privacy

The Extension is not designed for children under 13 and does not knowingly collect any data from children.

9. Updates to This Policy

We may update this Privacy Policy occasionally. The updated version will always be available on this page.
